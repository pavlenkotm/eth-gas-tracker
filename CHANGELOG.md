# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-11-07

### Added

#### Core Project
- Python ETH Gas Tracker with multi-network support
- Web UI dashboard for gas price monitoring
- REST API server for programmatic access
- Real-time alerts and notifications

#### Multi-Language Examples
- **Solidity**: ERC-20 token and ERC-721 NFT implementations
- **Vyper**: Simple vault contract with ApeWorX
- **Rust**: Solana counter program with Anchor framework
- **Move**: Aptos fungible token implementation
- **TypeScript**: Wagmi DApp with React frontend
- **Go**: Web3 utilities and RPC client
- **Java**: Web3j Ethereum integration
- **Bash**: Deployment automation scripts
- **HTML/CSS**: Professional DApp landing page

#### Documentation
- Comprehensive README for each language
- Contributing guidelines
- Code of Conduct
- Individual README files for all examples

#### CI/CD
- Python testing workflow (multi-OS, Python 3.8-3.12)
- Solidity/Hardhat testing workflow
- Rust/Anchor testing workflow
- TypeScript testing workflow
- Go testing workflow (Go 1.20-1.22)
- Java/Maven testing workflow
- Linting workflow (Markdown, YAML, Shell, Prettier)
- Dependabot configuration for automated updates

### Changed
- Repository restructured for multi-language support
- README updated with comprehensive overview

### Fixed
- Various bug fixes and improvements across all examples

## [1.0.0] - Previous Version
